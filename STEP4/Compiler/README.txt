Grading_Script_Step4.sh:
	1: Your grading shell script cannot open other shell script files
	   so to run it with .jar file use this Grading_Script_Step4.sh
	   
	   Instead of line 11: ./Micro $i > usertest/$output
	   Use:			     : java -jar MicroCompiler.jar $i > usertest/$output
	   
How to run MicroCompiler.jar in Windows/Linux:
	1: java -jar MicroCompiler.jar file_name.micro
	2: file_name.micro must be in the same directory

How to run shell script Micro.sh in Linux:
	1: ./Micro.sh file_name.micro
	2: file_name.micro must be in the same directory
	3: This shell script just run MicroCompiler.jar and takes argument from the command line,
	   it will not compile java classes
	   
	--- IF YOU WANT TO COMPILE IT ---

MicroCompiler.jar structure:
	MicroCompiler.jar is the Java executable program;
	   MicroCompiler.jar includes:
			MicroGrammar.g4,
			MicroListener.java,
			MicroExpression.java,
			Statement.java,
			CustomToken.java,
			MicroSymbolTable.java,
			GenerateStatement.java,
			VarDeclaration.java,
			CodeGeneration.java,
			DriverSymbolTable.java,
			MicroParser.java,
			MicroTokenizer.java,
			Driver.java,
			and all files which were generated by ANTLR tool.
